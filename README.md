# Portfolio: Esther Morales | Locutora Profesional & Voiceover

Este es el sitio web profesional de **Esther Morales**, locutora y especialista en voiceover. Un portfolio moderno, rápido y visualmente impactante diseñado para destacar su talento vocal y experiencia en la industria.

## 🚀 Características
- **Audio Interactivo**: Reproductor de demos integrado con WaveSurfer.js para una experiencia de escucha dinámica.
- **Diseño Premium**: Interfaz oscura y elegante con animaciones suaves y micro-interacciones.
- **Optimización Astro**: Rendimiento máximo con carga diferida de componentes interactivos.
- **Responsive**: Totalmente adaptado para dispositivos móviles, tablets y escritorio.

## 🛠️ Tecnologías
- **Framework**: [Astro 6.0](https://astro.build/)
- **UI**: [React](https://react.dev/) (para componentes interactivos de audio)
- **Estilos**: [Tailwind CSS 4.0](https://tailwindcss.com/)
- **Audio**: [WaveSurfer.js](https://wavesurfer-js.org/)
- **Iconos**: [Lucide React](https://lucide.dev/)

## 📂 Estructura del Proyecto

```text
/
├── public/              # Archivos estáticos y audios de demostración
├── src/
│   ├── assets/          # Imágenes y recursos multimedia
│   ├── components/      # Componentes Astro y React (UI)
│   ├── layouts/         # Plantillas de página (Layout.astro)
│   ├── pages/           # Rutas del sitio (index.astro)
│   ├── styles/          # Estilos globales (global.css)
│   └── content.config.ts # Configuración de Content Layer (Astro 6)
└── package.json
```

## 🧞 Comandos

Todos los comandos se ejecutan desde la raíz del proyecto:

| Comando | Acción |
| :--- | :--- |
| `pnpm install` | Instala las dependencias |
| `pnpm dev` | Inicia el servidor de desarrollo en `localhost:4321` |
| `pnpm build` | Compila el sitio para producción en `./dist/` |
| `pnpm preview` | Previsualiza la compilación localmente |
| `pnpm astro check` | Ejecuta comprobaciones de tipos y diagnóstico |

## ✨ Secciones del Sitio
1. **Inicio (Hero)**: Introducción profesional y llamada a la acción.
2. **Sobre Mí**: Trayectoria y especialidades de Esther.
3. **Demos**: Sección interactiva para escuchar muestras de voz.
4. **Servicios**: Detalle de servicios de locución y voiceover.
5. **Testimonios**: Opiniones de clientes y colaboradores.
6. **Contacto**: Información para contratación y consultas.
